# Data-Analytics-Portfolio

A portfolio of business, retail, and consumer analytics projects built using Python, SQL, Excel, and Tableau.

## Projects

### 1. Luxury Market Analysis
Analyzing luxury brand demand, consumer trends, and market performance.

### 2. Retail Sales Dashboard
Interactive dashboard exploring sales trends, top-performing products, and customer behavior.

### 3. Consumer Behavior Analysis
Examining purchase patterns and customer decision-making using retail transaction data.

### 4. Customer Segmentation Project
Segmenting customers based on purchasing behavior using RFM analysis and clustering.

### 5. Product Pricing Strategy Analysis
Investigating pricing patterns, category performance, and pricing opportunities in retail data.
